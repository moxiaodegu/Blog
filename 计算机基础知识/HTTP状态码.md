# 分类
 - 1xx 等待请求者继续请求
 - 2xx 成功
 - 3xx 重定向
 - 4xx 客户端错误
 - 5xx 服务器错误
# 常见状态码
 - 101 切换协议
 - 201 成功请求并创建了新资源
 - 202 成功接受请求，但未完成处理
 - 203 非授权信息，返回的meta信息不在原始服务器，而在一个副本
 - 204 无内容
 - 301 永久重定向
 - 302 临时重定向 /307 类似302 .不会从post变为get
 - 303 类似302，规定只能使用get
 - 304 协商缓存
 - 305 使用代理
 - 400 服务器无法解析
 - 401 身份认证
 - 404 找不到资源
 - 500 服务器内部错误
 - 502 无效响应
 - 504 网关问题
