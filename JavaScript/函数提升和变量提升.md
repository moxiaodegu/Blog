# 什么是提升
  > js代码在执行前，js执行引擎会进行预编译，预编译期间会将变量声明和函数声明提升到对应作用域的最顶端
### 函数提升
 - 函数提升只提升函数声明，不提升函数表达式
 - 函数声明式会提升到作用域最前边，并且将声明内容一起提升到最上边。

### 变量提升
 - 变量的声明都将被提升到作用域顶端。
   ```javascript
		console.log(v1);
		var v1 = 100;
		function foo() {
			console.log(v1);
			var v1 = 200;
			console.log(v1);
		}
		foo();
		console.log(v1);
		// undefined  undefined 200  100
   ```
